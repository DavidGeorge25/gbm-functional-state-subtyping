Abdelfattah cohort, GSE182109 / Broad SCP1985

Author

David George

Published

June 24, 2026

Introduction

External validation of the Mikolajewicz Iavarone subtyping analysis on the Abdelfattah et al. GBM scRNA-seq cohort (GSE182109, SCP1985).

Primary question: does PYGL enrichment in GPM tumour cells replicate at the patient level?

PYGB is a separate secondary hypothesis and is not pooled with PYGL.

The cohort mixes primary and recurrent Grade IV GBM (Type column). We report that mix as a limitation; it also asks whether any GPM/PYGL signal holds across disease settings.

Paired patient-level test

glioma_per_patient <- obj_tumor@meta.data %>%
  dplyr::count(Patient, name = "n_glioma")

patient_paired <- expr_long %>%
  dplyr::group_by(Patient, gene) %>%
  dplyr::summarise(
    gpm_mean = mean(expression[iavarone_state == "GPM"], na.rm = TRUE),
    rest_mean = mean(expression[iavarone_state != "GPM"], na.rm = TRUE),
    n_gpm = sum(iavarone_state == "GPM"),
    n_rest = sum(iavarone_state != "GPM"),
    .groups = "drop"
  ) %>%
  dplyr::inner_join(glioma_per_patient, by = "Patient") %>%
  dplyr::filter(n_gpm > 0, n_rest > 0) %>%
  dplyr::mutate(paired_diff = gpm_mean - rest_mean)

run_paired_stats <- function(pp_df) {
  lapply(present_targets, function(g) {
    sub <- pp_df %>% dplyr::filter(gene == g)
    if (nrow(sub) < 3) {
      return(tibble::tibble(
        gene = g, n_patients = nrow(sub), median_paired_diff = NA_real_,
        fold_change_gpm_vs_rest = NA_real_, wilcox_p = NA_real_, ttest_p = NA_real_,
        n_patients_gpm_higher = NA_integer_, verdict = "Insufficient patients"
      ))
    }
    wt <- wilcox.test(sub$gpm_mean, sub$rest_mean, paired = TRUE, exact = FALSE)
    tt <- t.test(sub$gpm_mean, sub$rest_mean, paired = TRUE)
    tibble::tibble(
      gene = g,
      n_patients = nrow(sub),
      median_paired_diff = median(sub$paired_diff),
      fold_change_gpm_vs_rest = mean(sub$gpm_mean) / mean(sub$rest_mean),
      wilcox_p = wt$p.value,
      ttest_p = tt$p.value,
      n_patients_gpm_higher = sum(sub$paired_diff > 0),
      verdict = if (median(sub$paired_diff) > 0 && wt$p.value < 0.05) {
        "Enrichment in GPM supported"
      } else {
        "Enrichment in GPM not supported"
      }
    )
  }) %>% dplyr::bind_rows()
}

Limitations

We had 17 patients in the paired test (one lacked GPM or non-GPM cells). Two patients contributed very few Glioma cells (ndGBM-10: 1, ndGBM-11: 21); the sensitivity analysis excluding patients with fewer than 50 Glioma cells is reported above. Primary and recurrent tumours are mixed in Type. CopyKat called ~29% of Glioma cells diploid; we used author Assignment labels only. Non-GPM cells are NEU-heavy, so the pooled rest mean is not MTC-specific. PPR signatures may miss H2AZ1/H4C3 as in Mikolajewicz. States are assigned winner-take-all from continuous scores. Pseudobulk avoids single-cell dropout in the headline figure.

Summary

PYGL is higher in GPM at the patient level in both cohorts; PYGB does not show clear GPM enrichment in either. Treat them as separate targets.
